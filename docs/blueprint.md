# **App Name**: Digital Blessing Chain

## Core Features:

- Blessing Creation and Minting: Allows users to write, categorize, and mint blessings as NFTs, automatically generating glow animations and aura colors based on sentiment, with gifting functionality.
- Blessing Chain Viewer: Displays a chronologically ordered chain of blessings with aura glow, creator info, timestamps, and links to NFT details.
- Interactive Explore Page: Provides a grid view of blessings, sorted by most uplifting, gifted, recent and by category, and feature soft hover animations and floating sparkles for a visually engaging experience.
- Generative Blessing Preview: Generates a glowing visual representation of the blessing text that the user inputs; serves as a tool that helps the user determine if they want to mint their blessing as an NFT.

## Style Guidelines:

- Primary color: Warm gold (#FFD700) to evoke a divine, spiritual feel.
- Background color: Very light desaturated gold (#FAF7E8) to create a soft, comforting backdrop.
- Accent color: Pale Yellow (#FAF884), slightly different hue from primary color, brighter and less saturated, for interactive elements.
- Body and headline font: 'Alegreya', a humanist serif with an elegant and intellectual feel
- Use gentle, glowing icons to represent categories and actions.
- Rounded cards with aura glow effects to present blessing tokens in a visually appealing manner.
- Subtle Framer Motion animations for transitions and interactions, enhancing the sense of peace and divinity.
